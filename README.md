# 🎉 Event Registration System

A simple web application built using Flask and SQLite to manage event registrations.

---

## 🚀 Features
- 👤 Create and manage users  
- 📅 Create and manage events  
- 📝 Register users for events  
- 📊 View all participants  
- 🔄 Full CRUD operations (Create, Read, Update, Delete)

---

## 🛠️ Tech Used
- Python (Flask)
- SQLite
- HTML, CSS

---

## ▶️ How to Run

1. Install dependencies  
pip install -r requirements.txt  

2. Run the app  
python app.py  

3. Open in browser  
http://127.0.0.1:5000/

---

## 📌 Note
This project is made for learning basic backend development and database operations.
